I want to show you how I built this website. It uses a Jekyll theme and is hosted on GitHub Pages. I got this theme from this GitHub link [Jekyll Paper Github](https://github.com/ghosind/Jekyll-Paper-Github).

### What You Need to Do  


1. Fork the GitHub repository of the theme.  
2. Rename the repository to `your_github_username.github.io`.  
3. On GitHub, navigate to your site's repository.  
4. Under your repository name, click `Settings`. If you cannot see the "Settings" tab, click the dropdown menu, then select `Settings`.  
5. In the `Code and automation` section of the sidebar, click `Pages`.  
6. Under `Build and deployment`, go to `Source` and select `Deploy from a branch`.  
7. Under `Build and deployment`, use the branch dropdown menu and select a publishing source. Choose `master`. 
8. Optionally, use the folder dropdown menu to select a folder for your publishing source.  
9. Click `Save`.  
10. Enter `your_github_username.github.io` in your browser to view your site.
    

If you stuck, you can go here and read:  
[Configuring a publishing source for your GitHub Pages site](https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site).
